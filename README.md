## Projects

### 1. Subscription Cost Calculator

**Summary:** A subscription tracking app that helps you monitor how much you spend on recurring services. Add subscriptions (Netflix, Spotify, ChatGPT Plus, etc.), set monthly or annual billing, and view your total monthly spend, annual cost, and daily cost. Supports both USD and INR currencies. Data is persisted in localStorage.

**Demo:** [Live Demo](https://subscription-cost-calc.vercel.app) · **Run locally:** `cd subscription-cost-calc && npm run dev` → [http://localhost:3000](http://localhost:3000)
